# 瞬忆 · AI 碎片记忆 Demo

## 项目简介

「瞬忆」是一款手机端 AI 碎片记忆管理 APP 前端 Demo。当前版本只做页面和交互原型，用前端 mock 数据模拟语音、灵感、剪贴板、待办和学习碎片的整理流程。

Demo 重点展示：

- AI 修正前后对比
- 自动分类与自动归档
- 记忆搜索与标签筛选
- 可回溯的记忆时间线
- 隐私控制入口
- 硬件未来扩展入口

当前项目为纯静态 HTML/CSS/JS，不依赖 Node/npm，不需要后端服务。

## 页面模块

### 首页

首页用于路演第一屏展示，强调产品视觉冲击力。

包含内容：

- 顶部品牌区：AI 碎片记忆
- 主视觉状态卡片：AI 正在整理今日记忆
- 今日有效记忆、自动归档、AI 修正质量
- 功能入口：语音转写、AI 修正、自动归档、隐私控制
- AI 修正展示卡片
- 今日重点记忆列表

### 记忆舱

记忆舱是核心展示页。

包含内容：

- 搜索框
- 标签筛选：全部、语音、文字、剪贴板、待办、灵感
- 核心能力区：AI 修正前、AI 修正后、自动归档、可回溯
- AI 实时处理台：转写中、修正中、归档中
- 分类统计
- 记忆列表
- 原始碎片与 AI 修正内容对比
- 自动标签、自动归档状态
- 点击卡片展开详情

重要程度已改为 10 分制展示，例如 `9.4/10`。

### 我的

我的页保持简洁，只保留基础功能。

包含内容：

- 个人信息卡片
- 记忆资产统计
- 设备状态：手机 APP、云端 AI
- 硬件未来扩展入口
- 隐私控制：一键暂停、本地加密、片段删除、低耗模式
- 记忆时间线
- 设置入口

## 视觉风格

设计关键词：

- 高级
- 未来感
- AI
- 记忆
- 科技
- 轻量
- 手机 APP

视觉规则：

- 主色为深蓝、紫色、白色、浅灰蓝
- 页面背景使用深蓝到浅灰蓝渐变
- 首页主卡片使用渐变背景
- 卡片使用大圆角和柔和阴影
- 标题加粗，重点数字突出
- 图标统一为线性科技风格
- 底部导航固定在页面底部
- 优先适配 375px 到 430px 手机宽度

## 隐私与安全边界

当前 Demo 不实现任何真实敏感能力。

明确不包含：

- 真实后台录音
- 麦克风权限申请
- 真实剪贴板读取或监听
- 键盘输入捕获
- 后台常驻
- 蓝牙连接
- 硬件真实接入
- 静默采集
- 后端数据上传

页面中的“演示采集”“剪贴板模拟”“隐私控制”“硬件终端”均为前端展示，不触发系统权限。

## 交互说明

- 底部导航可切换：首页、记忆舱、我的
- 记忆舱标签可筛选 mock 数据
- 记忆舱搜索框可按标题、内容、标签、地点搜索
- 点击记忆卡片可展开详情
- 点击功能入口或隐私按钮会显示演示提示，不执行真实系统操作
