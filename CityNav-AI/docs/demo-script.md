# Demo 演示脚本

## 演示目标

用 2 到 3 分钟展示“从打开页面到完成一次校园导览”的完整闭环。

## 演示步骤

1. 打开首页
   - 说明页面是手机端 H5，无需下载安装。
   - 指出顶部的起点、终点选择和校园地图。

2. 生成默认路线
   - 页面默认展示“校门口到活动中心”的路线。
   - 说明系统会自动计算路线，并生成文字导览。

3. 使用快捷路线
   - 点击“技能节报到”。
   - 展示地图上的高亮路线、起点标记和终点标记。

4. 查看 AI 文案
   - 展示预计步行时间、模拟距离、途经地标和分段步骤。
   - 说明当前为模拟 AI 文案，后续可接入真实大模型。

5. 播放语音导览
   - 点击“播放语音导览”。
   - 说明使用浏览器 Web Speech API，无需额外音频文件。

6. 完成目的地打卡
   - 点击“到达活动中心并打卡”。
   - 展示打卡进度变化。

## 结束语

城职灵导把校园地点数据、路线生成、地图可视化和语音导览合在一个轻量 H5 页面中，可以作为迎新、校园开放日和技能节活动的统一导览入口。
