# 产品功能文档

## 产品定位

城职灵导 CityNav AI 是面向新生、访客和校园活动参与者的轻量级校园导览工具。当前版本以 H5 Demo 形式验证核心体验：选择目的地、生成路线、查看地图高亮、听取语音导览。

## 目标用户

- 新生：快速熟悉校门口、教学楼、食堂、图书馆等高频地点。
- 访客：减少入校后的问路成本，快速到达活动现场。
- 活动参与者：围绕技能节报名、路演、作品展示等场景完成路线引导。

## 核心流程

1. 用户打开手机端页面。
2. 选择起点和终点，或点击快捷路线。
3. 系统基于地点路径图生成最短路线。
4. SVG 地图同步高亮路线，并标出起点和终点。
5. 页面生成模拟 AI 导览文案与分段步骤。
6. 用户可播放语音导览，到达后完成目的地打卡。

## 功能模块

- 地点数据模块：维护地点名称、类型、说明、坐标和图标。
- 路径图模块：维护地点之间的可通行路径、距离、时间和描述。
- 路线生成模块：使用 Dijkstra 算法计算模拟最短路线。
- 地图展示模块：使用 SVG 自适应显示校园简图和路线高亮。
- AI 文案模块：根据路线节点、路径说明和目的地提示生成导览说明。
- 语音导览模块：使用 Web Speech API 播放中文路线说明。
- 打卡模块：通过 `localStorage` 保存用户已点亮的目的地。

## 移动端体验要求

- 触控按钮高度不低于 48px。
- 表单、地图、结果卡片在小屏幕下单列显示。
- SVG 地图保持 1:1 比例并自适应容器宽度。
- 文本自动换行，避免按钮、卡片和地图标签溢出。

## 可扩展方向

- 接入真实校园地图、楼栋室内地图和实时位置。
- 支持用户画像，例如新生、家长、参赛学生、访客。
- 支持活动日程、拥堵提醒、无障碍路线和多语言语音。
- 接入大模型 API，生成更自然的个性化导览。
