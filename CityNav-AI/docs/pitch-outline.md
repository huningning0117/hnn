# 路演大纲

## 1. 开场

- 项目名称：城职灵导 CityNav AI。
- 一句话介绍：一个把校园地图、AI 路线说明和语音导览结合起来的手机端校园导航助手。

## 2. 问题背景

- 新生和访客第一次到校时，经常不知道教学楼、食堂、活动点之间怎么走。
- 校园活动期间，临时服务点多、路线变化快，传统静态指示牌不够灵活。
- 手机端轻量工具比下载 App 更适合短时、高频、现场使用。

## 3. 解决方案

- 使用 H5 页面快速打开。
- 通过地点选择和快捷场景生成路线。
- 用 SVG 地图高亮路线，降低理解成本。
- 用模拟 AI 文案和语音导览提供更自然的引导。

## 4. Demo 展示

- 展示默认路线：校门口到活动中心。
- 点击“技能节报到”快捷路线。
- 查看地图起点、终点和路线高亮。
- 播放语音导览。
- 到达目的地后完成打卡。

## 5. 技术亮点

- 移动优先响应式布局。
- 自绘 SVG 校园地图，自适应不同屏幕。
- JavaScript 路径图和 Dijkstra 路线生成。
- Web Speech API 中文语音导览。
- 静态站点可直接部署到 GitHub Pages。

## 6. 未来规划

- 接入真实校园数据和定位。
- 接入大模型 API 生成个性化路线说明。
- 打通活动系统、二维码定位和校内通知。
- 拓展为校园迎新、赛事活动、访客接待的一体化导览入口。
