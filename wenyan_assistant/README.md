# 🌸 温言助手 (Wenyan Assistant)

情绪识别与友好表达助手 - 让沟通更温暖

## 📖 项目简介

温言助手是一款智能情绪识别与语气改写工具，帮助用户在聊天中识别带有情绪的词汇，并及时提醒换一种更友好的表达方式，避免因言辞不当造成不良后果。

### 核心功能

- **🔍 情绪检测**：自动识别文本中的情绪风险
- **✨ 语气改写**：将带有情绪的文本改写为友好表达
- **🎭 多场景支持**：职场、朋友、家庭三种场景
- **🚫 脏话过滤**：自动过滤不文明用语
- **💡 沟通建议**：提供优化沟通的建议

## 💡 使用示例

### 情绪检测

输入：`"必须今天完成！"`

输出：
```
情绪检测结果：
  ⚠️  检测到愤怒，命令情绪
  ⚠️  语气严重不当，必须改写
```

### 语气改写

输入：`"烦死了"` (朋友场景)

输出：
```
【朋友改写结果】
  原文：烦死了
  改写：我有点无奈啦哦～
  
  🔄 改写步骤：
    友好表达替换:
      '烦死了' → '我有点无奈啦'
```

## 🎨 情绪等级

| 等级 | 图标 | 说明 |
|------|------|------|
| 安全 | ✅ | 语气友好，可直接发送 |
| 轻微 | ⚠️ | 语气稍显生硬，建议微调 |
| 中等 | ⚠️ | 语气较为激烈，建议改写 |
| 严重 | ❌ | 语气严重不当，必须改写 |
| 违规 | 🚫 | 包含违规用语，严禁发送 |

## 🌟 场景说明

### 职场场景
- 语气：专业、礼貌
- 后缀：，有问题随时沟通～
- 适用：工作沟通、邮件、报告

### 朋友场景
- 语气：轻松、自然
- 后缀：哦～
- 适用：朋友聊天、社交媒体

### 家庭场景
- 语气：温柔、体贴
- 后缀：，好不好呀～
- 适用：家人沟通、亲密关系

## 🔧 扩展功能

### 添加自定义词库

编辑 `data/emotion_words.py` 文件，在相应字典中添加新的词汇：

```python
# 添加风险词
RISKY_WORDS = {
    "愤怒": [..., "你的新词"],
    ...
}

# 添加友好替换
FRIENDLY_REPLACE = {
    "你的新词": {
        "职场": "职场友好表达",
        "朋友": "朋友友好表达",
        "家庭": "家庭友好表达"
    }
}
```
